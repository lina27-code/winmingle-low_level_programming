# 0x0B. C - malloc, free

Dynamic memory allocation in C: using `malloc` and `free` to manage heap memory.

## Concepts
- Stack vs heap
- Allocating and freeing memory
- Handling allocation failures
- Avoiding memory leaks
- Building 2D arrays dynamically
- Splitting strings into words

## Files
- `main.h` – function prototypes
- `0-create_array.c` – allocate and fill array
- `1-strdup.c` – duplicate string
- `2-str_concat.c` – concatenate two strings
- `3-alloc_grid.c` – allocate 2D integer grid
- `4-free_grid.c` – free 2D grid
- `100-argstostr.c` – combine arguments into one string
- `101-strtow.c` – split string into words

## Compilation
All files compiled with: gcc -Wall -Werror -Wextra -pedantic -std=gnu89 [source files] -o output
