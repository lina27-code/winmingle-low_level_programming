# 0x0E - Structures, typedef

This project introduces C structures and the `typedef` keyword.

## Files
- `dog.h` – defines struct dog and typedef dog_t
- `1-init_dog.c` – initializes a dog structure
- `2-print_dog.c` – prints a dog's details
- `4-new_dog.c` – creates a new dog (allocates memory)
- `5-free_dog.c` – frees a dog created by new_dog

## Compilation
All files compiled with: gcc -Wall -Werror -Wextra -pedantic -std=gnu89
