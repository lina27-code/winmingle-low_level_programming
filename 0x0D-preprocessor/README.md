# 0x0D - C Preprocessor

This project demonstrates the use of macros and include guards.

## Files
- `0-object_like_macro.h` – defines SIZE macro
- `1-pi.h` – defines PI macro
- `2-main.c` – prints its own file name
- `3-function_like_macro.h` – defines ABS macro
- `4-sum.h` – defines SUM macro

## Compilation
All programs compiled with: gcc -Wall -Werror -Wextra -pedantic -std=gnu89
