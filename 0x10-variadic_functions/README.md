# 0x10 - Variadic Functions

This project demonstrates variadic functions in C using `stdarg.h`.

## Tasks
- `0-sum_them_all.c` – sum all parameters
- `1-print_numbers.c` – print numbers with separator
- `2-print_strings.c` – print strings with separator, `(nil)` for NULL
- `3-print_all.c` – print anything based on format string

## Compilation
All files compiled with: gcc -Wall -Werror -Wextra -pedantic -std=gnu89


## Allowed functions
- `malloc`, `free`, `exit`
- `printf` (only where specified)
- `va_start`, `va_arg`, `va_end`
- `_putchar`
