# 0x0F - Function Pointers

This project demonstrates how to use pointers to functions in C.

## Tasks
- `0-print_name.c` – prints a name using a function pointer
- `1-array_iterator.c` – executes a function on each element of an array
- `2-int_index.c` – searches for an integer using a comparison function
- `3-calc.h`, `3-op_functions.c`, `3-get_op_func.c`, `3-main.c` – simple calculator
- `100-main_opcodes.c` – prints opcodes of its own main function

## Compilation
All files compiled with: gcc -Wall -Werror -Wextra -pedantic -std=gnu89:
