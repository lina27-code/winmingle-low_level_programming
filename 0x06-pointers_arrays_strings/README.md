# 0x06. Pointers, Arrays and Strings

## Description
This project is part of the WinMingle Software Engineering Training Program.  
It focuses on deeper understanding and practical implementation of **pointers, arrays, and strings in C programming**.

Students implement several functions that manipulate strings and arrays without using the standard C library.

## Learning Objectives
By completing this project, we learn how to:

- Use pointers effectively
- Work with arrays in C
- Understand the relationship between pointers and arrays
- Manipulate strings manually
- Understand variable scope

## Requirements
- Allowed editors: vi, vim, emacs
- Compiled on Ubuntu 20.04 using:
- No global variables
- No more than 5 functions per file
- No use of standard library functions like `printf` or `puts`
- Allowed function: `_putchar`
- All function prototypes are declared in `main.h`

## Files
| File | Description |
|-----|-------------|
| 0-strcat.c | Concatenates two strings |
| 1-strncat.c | Concatenates two strings with a limit |
| 2-strncpy.c | Copies a string |
| 3-strcmp.c | Compares two strings |
| 4-rev_array.c | Reverses an array of integers |
| 5-string_toupper.c | Converts lowercase letters to uppercase |
| 6-cap_string.c | Capitalizes words in a string |
| 7-leet.c | Encodes a string into leet speak |
| 100-rot13.c | Encodes a string using ROT13 |
| 101-print_number.c | Prints an integer using `_putchar` |
