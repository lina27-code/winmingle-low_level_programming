# 0x07-pointers_arrays_strings

This project covers:
- Pointers to pointers
- Multidimensional arrays
- String manipulation functions
- Memory manipulation functions

## Tasks
0. _memset - Fill memory with constant byte
1. _memcpy - Copy memory area
2. _strchr - Locate character in string
3. _strspn - Get length of prefix substring
4. _strpbrk - Search string for any set of bytes
5. _strstr - Locate substring
6. print_chessboard - Print chessboard
7. print_diagsums - Print sum of diagonals
8. set_string - Set value of pointer to char
