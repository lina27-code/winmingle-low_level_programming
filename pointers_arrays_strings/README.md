# Pointers, Arrays and Strings

## Project Overview
This project is part of the **WinMingle Software Engineering Training Program**.  
It focuses on understanding and implementing concepts related to **pointers, arrays, and strings in C programming**.

Students will practice manipulating memory using pointers, working with arrays, and performing operations on strings without relying on standard library functions.

---

## Learning Objectives
By completing this project, students should be able to:

- Understand what **pointers** are and how they work
- Use **arrays** effectively in C programs
- Understand the **difference between pointers and arrays**
- Manipulate **strings using pointers**
- Understand **variable scope**

---

## Requirements
- Allowed editors: `vi`, `vim`, `emacs`
- Compilation environment: **Ubuntu 20.04 LTS**
- Compiler: **gcc**
- Compilation flags:

- All files must end with a new line
- Code must follow **Betty style**
- No **global variables**
- Maximum **5 functions per file**
- Standard library functions such as `printf` or `puts` are **not allowed**
- Only `_putchar` may be used

## Tasks
This project includes tasks that cover:

- Updating values using pointers
- Swapping integers
- String length calculation
- Printing and reversing strings
- Working with arrays
- Copying strings
- Converting strings to integers
- Generating random passwords
