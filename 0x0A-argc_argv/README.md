# 0x0A - argc, argv

This project introduces command-line arguments in C.  
We learn how to access and use `argc` and `argv` in `main()`.

## Tasks
- 0-whatsmyname.c – print program name
- 1-args.c – print number of arguments
- 2-args.c – print all arguments
- 3-mul.c – multiply two numbers
- 4-add.c – add positive numbers
- 100-change.c – compute minimum coins for change
