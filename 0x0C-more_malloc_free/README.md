# 0x0C. C - More malloc, free

Dynamic memory allocation using `malloc`, `free`, and `exit`.  
Reimplementation of `calloc` and `realloc`.  
Handling large numbers via string multiplication.

## Files
- `main.h` – function prototypes
- `0-malloc_checked.c` – safe allocation with exit on failure
- `1-string_nconcat.c` – concatenate strings with size limit
- `2-calloc.c` – allocate and zero memory
- `3-array_range.c` – create integer array from min to max
- `100-realloc.c` – reallocate memory block
- `101-mul.c` – multiply two large numbers as strings

## Compilation
All files compiled with: gcc -Wall -Werror -Wextra -pedantic -std=gnu89 [source files] -o output
