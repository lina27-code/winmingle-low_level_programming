# 0x03. Debugging

## Description
This project focuses on debugging C programs by identifying and fixing logical and syntax errors.

## Learning Objectives
- Understand what debugging means
- Identify and fix coding errors
- Interpret compiler error messages
- Apply logical debugging techniques

## Requirements
- Ubuntu 20.04 LTS
- GCC with flags: -Wall -Werror -Wextra -pedantic -std=gnu89
- All files end with a new line
- Code written using vi, vim, or emacs
